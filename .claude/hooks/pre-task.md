# hook: pre-task — Đọc ngữ cảnh trước khi viết code

Bắt buộc đọc theo thứ tự TRƯỚC mọi thay đổi code:
1. `.claude/CLAUDE.md`
2. `.claude/AI_RULES.md`
3. `.claude/MEMORY.md`
4. `.claude/PROJECT_SUMMARY.md`
5. Project Bible (`docs/Project-Bible/`) phần liên quan + spec Sprint hiện tại (`SPRINT_PLAN.md`)
6. `.claude/skills/fpt-connect/` (SKILL, RULES, ARCHITECTURE, DATABASE, API_GUIDE, UI_GUIDE, CODING_STYLE, TESTING, SECURITY, PERFORMANCE)

Sau khi đọc: xác nhận phạm vi sprint, mắt xích truy vết `FR→UC→API→DB→TC`, và file dự kiến chạm. Nếu mơ hồ → hỏi, không đoán.
